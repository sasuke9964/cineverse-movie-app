# CineVerse - Movie Recommendation App

## 📌 Overview

CineVerse is a modern movie discovery application built using React.js that allows users to explore movies, search for titles, and manage favorites using real-time API data.

## 🚀 Features

* Search movies by name
* Browse movies by categories (popular, top-rated)
* View movie posters, ratings, and details
* Add/remove movies to favorites
* Persistent favorites using localStorage
* Responsive and modern UI

## 🛠️ Tech Stack

* React.js
* JavaScript
* OMDB API
* HTML & CSS
* LocalStorage

## 🧠 Key Concepts Used

* React Hooks (useState, useEffect)
* API integration using fetch & async/await
* State management
* Conditional rendering
* Component-based architecture
